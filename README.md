## Git tutorial
### Configure commands git
```
git config —list
git config —global user.name
git config —global user.email
```

### Type objects of Git
Blob - file
Tree - derectory
Commit - saved version
Annotated tag 

### Main git commands
```
git status
git add <files>
git commit -m "<message>"
git log
git checkout <commit hash>
git checkout <branch name>
```
